# PyRogue

## Project Description
PyRogue is a roguelike dungeon crawler game built in Python. Players navigate through procedurally generated dungeons, battling monsters, collecting loot, and upgrading their characters as they strive to conquer the depths and discover hidden treasures.

## Features
- **Procedurally Generated Levels:** Each gameplay session offers a unique challenge with endless dungeon variations.
- **Character Classes:** Choose from various classes, each with unique abilities and playstyles.
- **Engaging Combat Mechanics:** Turn-based combat system that emphasizes strategy and skill.
- **Loot System:** Discover and equip powerful items to enhance your character's abilities.
- **Save and Load Functionality:** Resume your adventures at any time.

## Architecture
The game is structured using object-oriented programming principles, making it easy to extend and maintain. The architecture consists of:
- **Game Engine:** Handles the main game loop, event handling, and rendering.
- **Entity System:** Manages players, enemies, items, and other game entities.
- **Level Generation:** Contains algorithms for creating dungeon layouts and populating them with enemies and loot.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dom1nati0n/PyRogue.git
   cd PyRogue
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the game:
   ```bash
   python main.py
   ```

## Usage
Start the game from the command line, and follow the prompts to create a character and begin your adventure. Navigate the dungeon using the arrow keys and engage in combat using the space bar. Explore, gather loot, and aim to conquer the dungeon!

Experience the thrill of exploring dark dungeons filled with hidden enemies, loot, and surprises. PyRogue offers hours of gameplay with its challenging mechanics and the ever-changing dungeon environment. Will you uncover the secrets hidden within?
